    Projetos

    Desempenho de Vereadores

    Plataforma web que disponibiliza informações sobre o desempenho dos vereadores da cidade de São José dos Campos durante o atual mandato. O objetivo é oferecer aos eleitores dados claros e acessíveis que ajudem a tomar decisões informadas nas eleições municipais.

    Dashboard de Indicadores de Crescimento

    Dashboard de Indicadores de Crescimento para monitorar o impacto da plataforma e acompanhar o crescimento das empresas cadastradas no Helpnei, exibindo dados estratégicos para patrocinadores e investidores. E um sistema de captação de usuários interessados em patrocínio.

    Gerenciador de Tarefas

    Gerenciador de tarefas desenvolvido para centralizar e otimizar o fluxo de trabalho, contemplando criação de equipes e projetos, definição de tarefas por prioridade e responsáveis, quadros Kanban, quadro geral de atividades, métricas em um dashboard de produtividade e sincronização com o Google Calendar, garantindo organização, colaboração e entregas ágeis.

    Sistema Nuel Tech

    Este projeto é uma aplicação web que permite o Cadastro, Leitura, Atualização e Exclusão (CRUD) de produtos, com sistema de autenticação de usuário integrado. Desenvolvido como prática de integração entre front-end e back-end, o sistema possui uma interface intuitiva e funcional. Os usuários precisam se cadastrar e fazer login para acessar a área de gerenciamento de produtos.

    Sistema World Beauty

    Sistema completo de gerenciamento para o salão de beleza fictício World Beauty, desenvolvido para centralizar e facilitar as operações do estabelecimento. A aplicação permite cadastrar clientes, registrar produtos e serviços, controlar consumos realizados durante os atendimentos e gerar métricas e relatórios que auxiliam na tomada de decisões.

    Atlantis - Ocean Solutions

    Sistema completo de gerenciamento para o hotel/pousada/resort fictício Atlantis.
    A solução permite realizar o cadastro de clientes (hóspedes) tanto como titulares quanto como dependentes, gerenciar acomodações, registrar hospedagens e acessar listagens e consultas relacionadas a todas essas operações.
